# Data Structures & Algorithms Implementation (C++)

## 📌 Overview
This project is a C++ implementation of fundamental Data Structures and Algorithms, developed as part of the CO2003 - Data Structures and Algorithms course at Ho Chi Minh City University of Technology.

The project focuses on building core data structures from scratch and applying algorithmic techniques to solve computational problems efficiently.

---

## 🚀 Features
- Custom implementation of fundamental data structures
- Algorithm design and problem-solving techniques
- Focus on time and space complexity optimization
- Modular and reusable code structure
- Practical applications of DSA concepts

---

## 🧱 Implemented Concepts
- Arrays, Linked Lists
- Stacks and Queues
- Trees / Binary Trees (if applicable)
- Sorting Algorithms (e.g., Quick Sort, Merge Sort)
- Searching Algorithms (e.g., Binary Search)
- Recursion and Divide & Conquer

---

## 🛠️ Tech Stack
- C++ (C++11)
- Data Structures
- Algorithm Design & Analysis
